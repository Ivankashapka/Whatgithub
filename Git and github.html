Git и Github что это такое?


GitHub — это служба размещения в Интернете репозиториев Git, которые используются для хранения содержимого docs.microsoft.com.
В GitHub размещается основной репозиторий всех проектов. Из этого репозитория участники копируют свою работу.


Git - Git - это консольная утилита, для отслеживания и ведения истории изменения файлов, в вашем проекте.
Чаще всего его используют для кода, но можно и для других файлов. Например, для картинок - полезно для дизайнеров.
